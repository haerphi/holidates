# Welcome in holiday ! (or not...)
With this npm-package you'll know when you'll be in holiday (legally)!
## Setup
### Install
`npm install @haertphi/holidates`
### Test
Just put this line in your index file:
`const test = require('@haerphi/holidates');`
And run:
`node . france`
or
`node . france 2000`
